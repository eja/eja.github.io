# landing page & repository directory

This repository contains the source code for the official landing page of the **eja** organization, hosted at [eja.github.io](https://eja.github.io).

## Purpose
The website acts as a centralized directory to showcase eja's active public repositories, highlighting their primary programming languages and most recent updates.

## Technical details
- **Automated Directory:** Dynamically retrieves all public repositories belonging to the `eja` organization via the GitHub REST API.
- **Filtering:** Automatically excludes repository forks and the landing page repository itself to maintain a clean workspace.
- **Design:** Built with Bootstrap 5, featuring a responsive grid layout optimized for mobile, tablet, and desktop viewports.
- **Static Hosting:** Deployed automatically using GitHub Pages.
